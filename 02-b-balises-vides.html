Block-level élément Il existe deux types de balises html, le premier type est le block, ces éléments sont conçu pour contenir d'autres éléments block ou inline.

  • Ils prennent par défaut toute la largeur de la page
  • Ils commencent toujours une nouvelle ligne
  • Ils possèdent un "margin" haut et bas

Quelques exceptions sont à compter comme la balise "p" qui ne peut contenir que des éléments inline.

<div> ... </div> L'élément "div" est donc un élément de type block... c'est tout.
Il n'a volontairement aucune particularité, il servira en lui donnant des classes ou id afin de mettre en forme votre page via le CSS.

Inline élément Le second type est le inline élément :

  • Ne contient pas d'élément block
  • Ne commence pas de nouvelle ligne
  • Prend seulement la largeur qui lui est necessaire
  • N'as pas de margin haut et bas

<span> ... </span> L'élément "span" est un élément de type inline... sans autre effet ou règle.
De la même façon que pour "div" celui ci sera lié à des classes ou id permettant de faire de la mise en forme.

Il y a des "div" et "span" dans cette page, mais sans CSS, rien de particulier ne s'affiche.
